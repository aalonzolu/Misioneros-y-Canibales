# Misioneros y canibales!

Recreación del juego de lógica Misioneros y Canibales con python y pygame.

Requerimientos
-------------

 - Python 3
 - Pygame
`sudo pip install -r requirements.txt`

Ejecutar
-------------
`python main.py`

Teclas
-------------
- Mover barco = Direccionales derecha e izquierda
- Subir Misionero = m
- Bajar Misionero = n
- Subir Canibal = c
- Bajar Canibal = x
